# Rubik’s Cube Solver (Python GUI)

A simple **Python GUI application** that generates random Rubik’s Cube scrambles, solves them using the **Kociemba algorithm**, and visually displays both the **scrambled** and **solved** cube.

## Features

* 🎲 Random scramble generator
* ⚡ Fast solving using **Kociemba algorithm**
* 🧊 Visual display of scrambled and solved cubes
* 📊 Shows move count and solving time
* 🖥 Built with **Tkinter** and **PyCuber**

## Installation

Install required libraries:

```
pip install pycuber kociemba
```

## Run the Application

```
python gui.py
```

## Project Structure

```
gui.py              # Main GUI application
kociemba_solver.py  # Solver interface
utils.py            # Scramble generator
```

## Technologies

Python • Tkinter • PyCuber • Kociemba Algorithm
